# Facebook Clone

## Description
This project is a Facebook Clone created using HTML and CSS only. It replicates the modern interface of Facebook's home page, focusing on clean layout, responsive design, and accurate styling without using JavaScript or frameworks.

## Features
- Responsive design that works on different screen sizes
- Modern Facebook UI layout with header, sidebar, and feed
- Story section with colorful story circles
- Post creation area with interactive elements
- Sample posts with comments and engagement features
- Sponsored content and contacts sidebar

## Technologies Used
- HTML5
- CSS3
- Font Awesome Icons

## Project Structure
```
├── index.html          # Main HTML file
├── styles.css          # Stylesheet with all CSS
└── README.md           # This file
```

## How to Run
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Explore the Facebook Clone interface
